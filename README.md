# 📊 Analyse de salaires avec Pandas

## 📌 Description
Projet d’analyse de données réalisé en Python avec la bibliothèque Pandas.

Ce script permet d’explorer un dataset de salariés et d’en extraire des informations clés.

---

## 🚀 Fonctionnalités

- Calcul du salaire moyen
- Calcul du salaire maximum
- Filtrage des personnes de plus de 30 ans
- Moyenne des salaires par département
- Identification du département le mieux payé
- Identification de la/les personne(s) la/les mieux payée(s)

---

## 🛠️ Technologies utilisées

- Python 3
- Pandas

---
